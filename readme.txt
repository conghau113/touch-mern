Hướng dẫn chạy project
Project bao gồm 2 folder: server và client
-	Folder server chứa toàn bộ phần code backend của server của project
-	Folder client chứa toàn bộ phần code frontend của project
Sau đó mở  terminal hoặc command và chạy các dòng lệnh sau:
Chạy server: dùng npm để chạy 
-	cd server : để đi đến folder server
-	npm install: để cài đặt các package, module
-	npm start: để run server
-	sau đó terminal/command sẽ hiển thị 
	o	Listening at port 4000
	o	MongoDB connected
Chạy client: dùng pnpm để chạy 
-	cd client: để đi đến folder client
-	npm install pnpm: cài đặt pnpm để chạy dự án
-	pnpm install: để cài đặt package và module
-	pnpm start: để run client
-	sau đó terminal/command sẽ hiển thị
	o	http://localhost:5173/ 
-	ấn vào link sẽ chuyển đến trang web của dự án
